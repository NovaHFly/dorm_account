Работа с комнатами

Для добавления комнаты необходимо нажать на кнопку "Создать". Высветится диалог со следующими полями ввода:

  • Номер комнаты - собственно, номер комнаты
  • Тип комнаты - определяет, люди какого пола могут жить в этой комнате. Общий тип означает отсутствие ограничений
  • Количество мест - сколько всего людей могут жить в этой комнате. Можно выставить лимит на 0, тогда комната будет закрыта для новых жильцов

После ввода данных нужно нажать на кнопку "Сохранить". В главной панели окна отобразится карточка с данными по комнате.

Чтобы выбрать комнату для редактирования или удаления нужно нажать на круглую кнопку в углу карточки. Затем нажать на одну из активировавшихся кнопок

Редактирование: высветится тот же диалог, что и при создании, только поля ввода будут заполнены.

Удаление: высветится диалог с подтверждением удаления. При нажатии на кнопку Да информация о комнате будет безвозвратно удалена.

Присутствует возможность сортировки комнат по нескольким параметрам.

Карточки с комнатами отображаются с разным цветом в зависимости от параметров:

Оранжевый - Мужские комнаты
Жёлтый - Женские комнаты
Серый - Общие комнаты
Тёмно-фиолетовый - Выбранная комната
Ярко-красный - В комнате отсутствуют свободные места

При нажатии на кнопку "Жильцы" высветится окно со списком жильцов в комнате. Подробную справку по этому окну можно получить при нажатии f1 на экране со списком жильцов комнаты.

Вышеперечисленные функции продублированы в главном меню вверху окна. Также там можно найти следующие функции:

  • Смена пароля
  • Закрытие программы
  • Вывод справки
