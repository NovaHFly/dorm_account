Работа с жильцами

Для добавления жильца необходимо нажать на кнопку "Создать". Высветится диалог со следующими полями ввода:

  • ФИО жильца - Фамилия, имя, отчество жильца
  • Пол жильца - Собственно, пол жильца
  • Возраст жильца - Собственно, возраст жильца
  • Номер телефона - Контактный номер телефона жильца
  • Паспортные данные - Данные паспорта жильца

После ввода данных нужно нажать на кнопку "Сохранить". В главной панели окна отобразится строка с данными жильца (Кроме паспорта).

Чтобы выбрать жильца для редактирования или удаления нужно нажать на круглую кнопку в начале строки. Затем нажать на одну из активировавшихся кнопок.

Редактирование: высветится тот же диалог, что и при создании, только поля ввода будут заполнены.

Удаление: высветится диалог с подтверждением удаления. При нажатии на кнопку Да информация о жильце будет безвозвратно удалена.

Присутсвует возможность сортировки жильцов по нескольким параметрам и фильтрация жильцов по имени и полу.

В зависимости от пола жильца строки отображаются с разным цветом:

  • Оранжевый - Мужчина
  • Жёлтый - Женщина

Вышеперечисленные функции продублированы в главном меню вверху окна. Также там можно найти следующие функции:

  • Закрытие окна
  • Вывод справки
